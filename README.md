# 项目简介 python346

本项目为一款基于 Python 语言及 Django 框架开发的图书管理系统，旨在为图书管理相关业务提供便捷、高效的解决方案。

## 技术栈

- 后端：Python 3.x，Django
- 数据库：SQLite
- 前端：HTML/CSS/JavaScript

## 功能模块

### 核心功能

- **登录功能**：用户身份认证
- **图书管理**：图书信息的增删改查等操作
- **借阅管理**：借阅信息记录与管理，包含**续借功能**
- **个人管理**：用户个人信息管理
- **会员管理**：会员用户的高级管理功能
- **数据可视化**：图书借阅数据的可视化展示

### 系统角色

- 系统管理员
- 普通用户
- 会员用户

## 运行环境

- Python 3.x
- Django 2.x 或以上版本
- SQLite 数据库

## 部署步骤

（具体部署步骤将在使用文档中详细说明）

1. 环境准备：安装 Python 和 Django
2. 数据库配置：使用 SQLite
3. 应用部署：运行 Django 项目
